
# ftorrent

- **docs** (TBD) Documentation site _[docs.ftorrent.com](https://docs.ftorrent.com/)_
- **good** (Docker) IPv6 connection detector _[good.ftorrent.com](https://good.ftorrent.com/)_
- **notes** Notes and scraps
- **open** (Docker) BitTorrent and WebTorrent trackers _[open.ftorrent.com](https://open.ftorrent.com/)_
- **site** Home page and website _[ftorrent.com](https://ftorrent.com/)_
